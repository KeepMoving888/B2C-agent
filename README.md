# 跨境电商智能客服系统

## 🎯 项目简介

一个基于 LangGraph 的多 Agent 智能客服系统，专为跨境电商场景设计，支持亚马逊、Shopify 等多平台，提供订单、物流、售后全流程服务。

## ✨ 核心功能

- **多平台支持**：Amazon、Shopify、官网
- **多语言支持**：中文、英文、西班牙语、德语、法语、日语、泰语、越南语
- **智能路由**：自动识别用户意图，分配到对应专业 Agent
- **FAQ 秒回**：常见问题毫秒级响应
- **流式输出**：实时打字效果，提升用户体验
- **国内大模型**：支持 qwen 大模型作为 GPT 替代方案
- **性能优化**：响应时间 < 2 秒

## 🛠️ 技术栈

- **后端**：Python 3.11, Flask, LangGraph, LangChain
- **前端**：HTML5, CSS3, JavaScript
- **AI 模型**：支持 OpenAI, Anthropic, 国内大模型 (qwen)
- **向量数据库**：支持 ChromaDB, Pinecone, Milvus, FAISS
- **部署**：Docker, Kubernetes

## 📦 安装部署

### 1. 环境准备

```bash
# 克隆项目
git clone <repository-url>
cd B2C agent

# 创建虚拟环境
python -m venv venv

# 激活环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置

复制 `.env.example` 文件为 `.env` 并填写配置：

```bash
# 复制配置文件
cp .env.example .env

# 编辑配置
# 填写 qwen API 密钥等信息
```

### 3. 启动服务

```bash
# 启动开发服务器
python src/app.py

# 访问 http://localhost:5000
```

### 核心 Agent

- **咨询接待**：产品咨询、价格查询
- **订单履约**：订单状态、物流追踪
- **售后处理**：退换货、退款、投诉
- **合规解答**：平台政策、保修、A-to-Z 保障
- **人工转接**：复杂问题转人工

### 路由系统

- **快速路由**：基于关键词匹配
- **意图识别**：智能理解用户需求
- **平台适配**：根据选择的平台调整回复内容

### 工具集成

- **订单工具**：订单状态查询、订单详情获取
- **物流工具**：物流轨迹查询、时效预估
- **售后工具**：退换货标签、退款审批
- **用户工具**：用户信息查询、会话历史

### 当前版本优化

**注意**：为了保证回复速度（响应时间 < 2 秒），当前版本采用了以下优化：
- **简化 Agent**：保留多 Agent 架构设计，但暂时不使用工具调用功能
- **自动语言检测**：根据用户输入自动检测语言，无需手动选择
- **自动平台检测**：根据用户输入自动检测平台，无需手动选择
- **FAQ 秒回**：常见问题毫秒级响应
- **流式输出**：实时打字效果，提升用户体验

工具调用功能已在代码中保留，可根据需要随时启用。

## 🎨 自定义开发

### 添加新 Agent

1. 在 `src/agents/` 目录创建新的 Agent 文件
2. 在 `src/agents/graph.py` 中注册新 Agent
3. 更新路由规则

### 扩展知识库

1. 在 `data/knowledge_base/` 添加新文件
2. 运行 `python init_knowledge_base.py` 重新导入
